Projects

Projects done for Data Structures and Algorithms course.

Sequence

 A vector-like abstract data type that is implemented using a doubley-linked list. Supports element O(1) access, O(n) search, and two cases for removes and inserts: O(1) when inserting at head or tail, and O(n) for index inserts and removes.
Difficulty:

AVL Tree

 A self balancing BST that utilizes rotations. Does not support deletions. Supports O(logn) search, range search, and inserts. Overloads << operator to horizontally print the tree in O(n) time using inorder DFS.
Difficulty:

Alphabet Trie

 An autocompletion "tool" that uses a provided 62,000-word text file. Supports O(l) insert and search, where l is the number of letters in the word.
Difficulty:

Hashed Database

 A database implementation that uses an index table to acheive O(1) search and insert.
